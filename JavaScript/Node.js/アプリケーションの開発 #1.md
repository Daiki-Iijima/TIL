# Node.jsを使用してWebアプリケーションを作成 

## Node.jsのインストール状態の確認

```
node -v
```

## 初期化

```
npm init
```

- Jsonファイルを作成されるので今回はEnter連打で最後は「Yes」

- mainは「server.js」

- jsonの中身がこうなればOK

  ```json
  {
    "name": "local_node",
    "version": "1.0.0",
    "description": "",
    "main": "server.js",
    "scripts": {
      "test": "echo \"Error: no test specified\" && exit 1"
    },
    "author": "",
    "license": "ISC"
  }
  ```

## server.jsの中身

```javascript
var http = require("http");

//WebServerオブジェクト
var server = http.createServer();

//serverからのコールバック
server.on("request", function (req, res) {
    console.log("リクエストがあったよ");
});

//ポートを6000開いて
server.listen(6000);
```

# 現在のファイル構造

```
--- Local_Node
		|--- package.json
		|--- server.js
```
