# 継承とは？
- 元になるクラスのメンバーを引き継ぐこと
- 新たな機能の追加、元の機能を上書きしたりする仕組み
- 元になるクラスのことを基底クラス、スーパークラス、親クラスと呼ぶ
- 継承先のクラスのことを派生クラス、サブクラス、子クラスと呼ぶ

# 継承の仕方

```c#
[修飾子] class 派生クラス名 : 基底クラス
{
  ....派生クラスの定義.....
}
```

## 命名
- 派生クラスの名前は基底クラスよりも具体的につける
- 一般的に2単語以上で命名する
- 末尾に基底クラスの名前を付与するといい
  - 具体例
   - Exceptionクラスの派生クラスの場合
   - ArgmentException
   
## 継承時の制限
- 派生クラスが継承できる基底クラスは1つのみ
- これを単一継承という

```c#
public class SuperCar : Car,Parson //エラー
public class SuperCar : Car        //OK
```
