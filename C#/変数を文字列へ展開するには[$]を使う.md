# 文字列への変数展開

- 文字列の中に変数の値をそのまま表現したい場合に[$]を先頭につけ、文字列の中に{変数名}で埋め込めるようになる

```C#

//====例====
string name = "ラララ君";

Console.WriteLine($"私の名前は{name}です");
//結果 : 私の名前はラララ君です

//====逐語的文字列リテラルとのコンボ====

string name = "ラララ君";

Console.WriteLine($@"{name}は、
今日の朝に、
卵焼きを食べました");

//結果 :ラララ君は、
        今日の朝に、
        卵焼きを食べました
        

```
