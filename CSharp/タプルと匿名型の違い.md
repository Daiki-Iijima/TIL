# 匿名型とタプル型の違い

| 相違点     | 匿名型         | タプル       |
| ---------- | -------------- | ------------ |
| 型の分類   | 参照型(クラス) | 値型(構造体) |
| 変更の可否 | 不可           | 可           |
| 型宣言     | 利用不可       | 利用可       |

## タプル

- 引数として使用したい場合はタプル
- 変数・引数・戻り値などに使える

## 匿名型

- 匿名のオブジェクトのリテラルを表現するための仕組み
- 変数・引数・戻り値などに使えない
- 基本的にLINQでメンバーを切り出す時に使う
