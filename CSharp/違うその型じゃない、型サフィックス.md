# 前提
- 数値リテラルは指定がない場合、整数は**int**、浮動小数点数は**double**とみなされる

# 型サフィックスとは

- 明示的にデータ型を指定したい場合に使う
- リテラル型の末尾に接尾辞(サフィックス)を付与する

```C#
//デフォルト
var i = 100;  //int型
var d = 1.01; //double型

//上記をuint型とfloat型に変えたい
var i = 100u; //uint型
var d = 1.01f; //float型
```

# 型サフィックス表

| データ型    | サフィックス      | 例     | 
|---------|-------------|-------| 
| long    | l,L         | 100l  | 
| unit    | u,U         | 100u  | 
| ulong   | ul,UL,Ul,uL | 100ul | 
| float   | f,F         | 100f  | 
| double  | d,D         | 100d  | 
| decimal | m,M         | 100m  |
