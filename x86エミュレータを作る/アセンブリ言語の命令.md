# オペコード

- 定型的な処理を行う命令
- プログラム言語で言い換えると「関数名」であり、単語として動詞を用いている

# オペランド

- オペコードで処理をしたい「値」や「ポインタ」を指定する
- プログラム言語で言い換えると、関数に渡す引数

# ニーモニック

- 「オペコード」と「オペランド」をまとめた呼び方
- プログラム言語で言い換えると「関数と引数」(ニュアンス的にこれぐらいしか思いつかない)
