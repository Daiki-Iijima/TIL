# 変数の定義
## var
### 基本
型推論に任せる

```swift
var i = 1 
    ||
int i = 1
```

### 明示的に型を宣言

この定義をした場合は使用前までに初期化する必要がある

```swift
var i : int

print(i + 5) //Error(初期化してないので)

``` 

## let
 
定数として値を変えたく無い場合に使用する

ポインタのアドレスを固定するだけ

```swift
let i = 1

i = 2 //Error
```

# Swiftのコンストラクタ

クラスのプロパティを初期値を設定する

```swift

var value : Int

init()
{
  self.value = 1
}


```
