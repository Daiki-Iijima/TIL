# 今まで学んだタグ

| タグ     | 機能                                                         |
| -------- | ------------------------------------------------------------ |
| !DOCTYPE | ドキュメントタイプを宣言する<br /><!DOCTYPE html>文章がHTML5で作成されたものであるということを宣言する |
| html     | HTML文章であることを宣言する                                 |
| head     | 文章のヘッダ情報を表す                                       |
| body     | 文章の本体を表す                                             |
| h1〜h6   | 見出しをつける                                               |
| p        | 一つの段落(パラグラフ)であることを表す                       |

# CSS連携

| タグ | 機能 | 機能                                                  |
| ---- | ---- | ----------------------------------------------------- |
| link | ref  | リンク先のファイルを指定<br />例 : ref = "stylesheet" |
|      | href | CSSファイル(リソース場所)を指定する                   |
