## Gitの全体像

Gitのクローン後のローカルリポジトリの構造
 - 作業ディレクトリ
 - 「.git」ディレクトリ

- 作業ディレクトリ
 - gitで管理されるファイルが入っている

- .gitディレクトリ
 - gitリポジトリの本体
 - このディレクトリを目印にリポジトリ内のファイルを管理している

.gitファイルを移動させればgitのバージョン管理対象を変えることもできる
