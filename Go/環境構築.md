# Goの基本

## インストール方法

1. 公式のダウンロードページへGo

   1. URL <https://golang.org/dl/>

   2. Windowsだったら「.msi」を選んでおけばいいはず

      ![GoInstall](https://user-images.githubusercontent.com/40718695/56857638-36ba1680-69ab-11e9-939f-fac8e13b49ac.jpg)

2. インストールできているか確認

   1. コマンドラインからバージョン確認

      ```
      go version
      ```

   2. こんなのが出ればOK

      ```
      go version go1.12.4 windows/amd64
      ```
